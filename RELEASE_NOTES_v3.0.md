# SEKURY NETSUITE v3.0

Release inicial del proyecto unificado.

## Incluye

- Analisis de conexiones locales con puntaje de riesgo.
- Panel de amenazas con bloqueo, desbloqueo y gestion de procesos.
- Modulo NETMAPPER integrado para puertos, SO, servicios y vulnerabilidades.
- Build automatizado con EXE, ZIP, instalador NSIS y hashes SHA256.

## Artefactos esperados

- SEKURY_NETSUITE.exe
- SEKURY_NETSUITE_release.zip
- SEKURY_NETSUITE_Installer.exe
- SHA256SUMS.txt

## Notas

- Requiere Windows.
- Para el modulo NETMAPPER se necesita Nmap instalado en el sistema.
- Usar solo en sistemas y redes autorizadas.
